Live MtGox BTC exchange rates in xmobar

James Brotchie
08 April 2013

The bitcoin bubble is blowing up! and I have found it annoying keeping track of the BTCUSD and BTCAUD exchange rate on the MtGox website. Given that I'm already running xmobar wouldn't it be great if I could do this

and have the exchange rates update live.

To this end I've written xmobar-mtgox, a Python script that outputs xmobar friendly BTC exchange rates. Installation and usage instruction are on the GitHub project landing page.
